## Usage
Select a method below and get started with **witchCSS**!
### CDN (jsDelivr)
```
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/witchcss@2">
```

### npm
```
npm install witchcss
```

### Yarn
```
yarn add witchcss
```
